# Game Improvements

1. **Improve the music**
   - Rework the music so that it feels more polished, memorable, and appropriate for the game.

2. **Add accurate artwork for the main playable character**
   - Create character images that accurately represent the protagonist.
   - Display the protagonist's artwork during character dialogue so the player can see them while they are speaking.

3. **Redesign dialogue presentation**
   - Make dialogue take over the entire screen in a visual-novel style.
   - Replace the current small dialogue window with a more immersive presentation.

4. **Expand the dialogue**
   - Make conversations significantly longer and more detailed.
   - Use dialogue to properly explain events, characters, motivations, and relevant story information instead of rushing through them.

5. **Improve combat responsiveness and variety**
   - Make combat feel smoother and less clunky.
   - Add more attacks and combat options.
   - Make movement faster, quicker, and more powerful so that combat feels more energetic.
   - Apply similar improvements to enemy movement and attacks.

6. **Increase the difficulty**
   - Make enemies more challenging to fight.
   - Rebalance the game so that it does not feel too easy.

7. **Make upgrades more impactful**
   - Review every upgrade to confirm that it is actually applied during gameplay.
   - Verify that relevant values—such as damage, other stats, and critical-strike chance—are increasing correctly.
   - Strengthen or rebalance upgrades so that the player can clearly feel their impact.

8. **Add animated damage numbers**
   - Show damage numbers on screen whenever an enemy or the player takes damage.
   - Animate each number so that it quickly pops in, pops outward, and then disappears.

9. **Add special and charged attacks**
   - Introduce special attacks to create more combat variety.
   - Add a satisfying charged attack designed around the scythe as the main weapon.

10. **Redesign the main menu**
    - Give the main menu a more polished, distinctive, and game-like presentation.

11. **Strengthen the story and characters**
    - Expand the story so that it feels deliberate rather than rushed.
    - Improve character designs and give each character a clearer, more memorable personality.
    - Add a proper introduction and explain the world, characters, motivations, and events more thoroughly.
    - Address the current blandness by giving scenes and characters more depth.

12. **Establish a cohesive visual style**
    - Complete a full visual-design pass across the game.
    - Create and consistently apply a strong visual identity to the menus, dialogue, characters, combat feedback, and overall presentation.

13. **Misc**
    - Ensure that we have some sort of player statistics that we track too i.e., time spent on this run, time spent in total in this game, enemies killed, damage done total etc.
    - Make the last boss a lot faster and stronger, not just increase its HP numbers, though we need to do that as well, also make the boss more complex.
