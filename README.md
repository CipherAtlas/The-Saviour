The Saviour is a browser action roguelite about Zephyr, a scythe-wielding prince searching for the missing Princess Elowen. Guided by the bond between their paired rings, he must descend through ten dungeon floors, overcome the ancient Witch, and reach Elowen before she is lost.

## Download and play

1. Install [Node.js](https://nodejs.org/) if it is not already installed.
2. Select **Code → Download ZIP** on this repository, then extract the ZIP.
3. Open a terminal in the extracted folder and run:

```bash
npm install
npm run dev
```

4. Open the local address shown in the terminal.
